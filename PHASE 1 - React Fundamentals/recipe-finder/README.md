# 🍲 Recipe Finder

## 🚀 Features

- Search recipes using API
- View recipe details
- Loading & error handling

## 🧠 Approach

- Built using React functional components
- Managed state with useState
- Used useEffect for API calls
- Conditional rendering for UX states

## 🏗️ Architecture

- App → SearchBar → RecipeList → RecipeCard → RecipeDetails

## 🔌 API Used

- Spoonacular API
  
- ## ⚡ Challenges Faced

- Handling API loading states
- Managing empty results

## ⚙️ Setup

- npm install
- npm run dev
